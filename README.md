# 🌙 Guia de Tarot

Site pessoal de estudo e registro de tiragens de tarot.

🔗 **[Abrir o Guia](https://lumaminikel.github.io/tarot/)**

## Páginas

- **`index.html`** — Guia completo com todos os 78 arcanos, rituais, tiragens e combinações
- **`diario-tarot.html`** — Diário de tiragens com histórico salvo no navegador

## Sobre

Guia construído com HTML puro, sem dependências externas.  
O diário usa `localStorage` — os dados ficam salvos só no seu navegador.
